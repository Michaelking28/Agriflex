# Agriflex 

Agriflex is a frontend project I'm building using *React* and *Vite*. It's designed to be a clean, modern landing and onboarding experience for an agriculture-related platform.

## Project Status
This project is currently *in progress*, I’ve completed a few onboarding pages and basic navigation using React Router. I'm still actively working on it as I learn more about frontend development.

- CSS
